Minify done with:
* HTML -> [willpeavy](https://www.willpeavy.com/minifier/)
* JS -> [jscompress](https://jscompress.com/)

***

### "Features"
* Web Multiplayer
* Moving Left and Right (A & D)
* Jumping (Space), double jump
* Choose hat Color
* Walk and watch your hat grow
* Write Messages
* Project Size under 13 KB
